About this product

Product Information

Take it back to the late 80s with the Nintendo Game Boy. Adults and children love playing a variety of games on this system that can be taken almost anywhere. The hand-held Game Boy is perfect for playing in vehicles, on lunch breaks, or out in the backyard with friends. This system features a directional pad, as well as four operation buttons for in-game use. An external power supply jack allows users to plug the game into an AC power adapter. When users are away from a power source, the game runs on four AA batteries. A port lets players connect two Game Boys to one another, or the port can be used to connect the game to a Game Boy printer.
